BBQ Static Generator

Current status: Alpha

This is a static page generator written in Haskell. It is a library and you can start with the scaffolding site to build your blog with it.

Highlights

  • Based on Shake build system to boost performance
  • Utilizing template haskell and shakespeare library to enhance the templating capacity
  • Modular design
  • JSON widget interface

Major difference between it and Hakyll

Yes, Hakyll is much more stable and popular, but bbq-sg is not just another reinvented wheel. First, this library tried hard to use the third-party libraries which is more universal & mature, rather than reinventing a small scope of functionalities inside the library itself.
